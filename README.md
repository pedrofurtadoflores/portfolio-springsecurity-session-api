# Portfolio Spring Security Session API – Spring Boot

This is a portfolio project built with Java and Spring Boot, showcasing a secured RESTful CRUD API using session-based authentication with Spring Security. The application follows the MVC architecture (Model-View-Controller) and demonstrates clean code practices, DTO validation, Swagger documentation, and unit testing.

## Tech Stack

- **Java 17**
- **Spring Boot 3.2.5**
- **MVC Architecture**
- **Spring Security**
- **Spring Data JPA**
- **Bean Validation (Jakarta)**
- **Lombok**
- **PostgreSQL**
- **Swagger/OpenAPI 3.1 (via `springdoc-openapi-starter-webmvc-ui:2.5.0`)**
- **JUnit 5 & Mockito**
- **Maven**
